#NA Imputation Tool

R Shiny로 만든 데이터 결측치 처리 웹앱입니다.

##Live Demo
*(shinyapps.io 배포 후 링크 추가 예정)*

##주요 기능
- 컬럼별 결측치 현황 요약 (50개씩 페이지네이션)
- 히스토그램 + IQR 기반 이상치 자동 탐지 및 시각화
- Rule-based NA 처리 방법 자동 추천 (수락/거절)
- 수동 처리 + KNN / MICE 글로벌 대치 파이프라인
- 고운돋음 폰트 적용 커스텀 UI

##Tech Stack
- R Shiny
- DT
- VIM
- mice
